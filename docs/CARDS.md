### 마이너 아르카나 (Minor Arcana) - 56장

마이너 아르카나는 일상적인 상황과 구체적인 사건을 나타냅니다.

#### 🔥 지팡이 (Wands) - 14장
**원소**: 불 | **상징**: 열정, 창조, 행동, 영감

- 에이스~10: 열정의 시작부터 완성까지
- 시종(Page): 열정적인 메시지
- 기사(Knight): 빠른 행동과 모험
- 여왕(Queen): 자신감과 카리스마
- 왕(King): 리더십과 비전

#### 💧 컵 (Cups) - 14장
**원소**: 물 | **상징**: 감정, 사랑, 관계, 직관

- 에이스~10: 감정의 시작부터 성취까지
- 시종(Page): 감정적 메시지
- 기사(Knight): 로맨틱한 제안
- 여왕(Queen): 공감과 돌봄
- 왕(King): 감정적 균형

#### ⚔️ 검 (Swords) - 14장
**원소**: 공기 | **상징**: 생각, 논리, 갈등, 진실

- 에이스~10: 명확성에서 결론까지
- 시종(Page): 호기심과 탐구
- 기사(Knight): 결단력과 행동
- 여왕(Queen): 독립성과 통찰
- 왕(King): 지적 권위

#### 🪙 펜타클 (Pentacles) - 14장
**원소**: 흙 | **상징**: 물질, 재정, 건강, 현실

- 에이스~10: 기회에서 풍요까지
- 시종(Page): 새로운 기술 학습
- 기사(Knight): 꾸준한 노력
- 여왕(Queen): 실용적 돌봄
- 왕(King): 재정적 성공

## 카드 데이터 구조

```typescript
interface TarotCard {
    id: number;              // 고유 ID (0-77)
    name: string;            // 영문 카드명
    nameKr: string;          // 한글 카드명
    arcana: 'major' | 'minor';  // 아르카나 종류
    suit?: 'wands' | 'cups' | 'swords' | 'pentacles';  // 마이너 아르카나 수트
    upright: string;         // 정방향 키워드
    reversed: string;        // 역방향 키워드
    description: string;     // 카드 설명
    advice: string;          // 조언 메시지
    image?: string;          // 이미지 경로
}
```

## 카드 사용 규칙

### 스프레드별 카드 사용

| 스프레드 | 카드 수 | 사용되는 카드 |
|----------|---------|---------------|
| 원카드 | 1장 | 메이저 아르카나만 |
| 쓰리카드 | 3장 | 메이저 아르카나만 |
| 연애 스프레드 | 5장 | 전체 덱 (78장) |
| 선택 스프레드 | 5장 | 전체 덱 (78장) |
| 켈틱 크로스 | 10장 | 전체 덱 (78장) |

### 역방향 카드

카드 선택 시 자동으로 역방향 여부가 결정됩니다:
- 약 30% 확률로 역방향 출현
- 역방향은 카드의 에너지가 막혀있거나 반대 의미를 나타냄

## 카드 이미지

카드 이미지는 `/public/cards/` 디렉토리에 저장됩니다:
- 파일명: `{id}.png` (예: 0.png, 1.png, ...)
- 권장 크기: 400x600px
- 형식: PNG (투명 배경 권장)

## 커스터마이징

### 카드 데이터 수정

카드 데이터는 `src/data/tarotCards.ts`에서 수정할 수 있습니다:

```typescript
// 예: 바보 카드 수정
{
    id: 0,
    name: "The Fool",
    nameKr: "바보",
    arcana: "major",
    upright: "새로운 시작, 순수함, 모험",
    reversed: "무모함, 위험 경고, 망설임",
    description: "순수한 마음으로 새로운 여정을 시작합니다.",
    advice: "두려움 없이 첫 발을 내딛어보세요.",
    image: "/cards/0.png"
}
```

### 카드 이미지 교체

1. 새 이미지를 `/public/cards/` 폴더에 저장
2. 파일명을 해당 카드 ID로 지정 (예: `0.png`)
3. 앱을 새로고침하면 자동 적용
