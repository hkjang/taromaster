# 📊 스프레드 가이드

스프레드(Spread)는 타로 카드를 배치하는 패턴으로, 각 위치가 특정 의미를 가집니다.

## 사용 가능한 스프레드

### 🎯 원카드 (Single Card)

> 가장 간단한 스프레드로, 오늘의 메시지를 받습니다.

**카드 수**: 1장  
**추천 상황**: 빠른 조언, 오늘의 운세, 예/아니오 질문

```
   ┌─────┐
   │  1  │
   └─────┘
   오늘의
   메시지
```

**위치 의미**:
| 위치 | 의미 |
|------|------|
| 1 | 오늘의 메시지 |

---

### ⏰ 쓰리카드 (Three Card)

> 시간의 흐름을 따라 상황을 파악합니다.

**카드 수**: 3장  
**추천 상황**: 상황 분석, 결과 예측, 일반적인 질문

```
┌─────┐  ┌─────┐  ┌─────┐
│  1  │  │  2  │  │  3  │
└─────┘  └─────┘  └─────┘
 과거     현재     미래
```

**위치 의미**:
| 위치 | 이름 | 의미 |
|------|------|------|
| 1 | 과거 | 현재 상황의 원인, 지나온 경험 |
| 2 | 현재 | 지금 상황, 현재의 에너지 |
| 3 | 미래 | 앞으로의 방향, 가능한 결과 |

---

### 💕 연애 스프레드 (Love Spread)

> 연애 관계와 두 사람의 마음을 분석합니다.

**카드 수**: 5장  
**추천 상황**: 연애 상담, 관계 분석, 짝사랑

```
    ┌─────┐     ┌─────┐
    │  1  │     │  2  │
    └─────┘     └─────┘
    나의 마음   상대방의 마음
    
         ┌─────┐
         │  3  │
         └─────┘
         현재 관계
         
    ┌─────┐     ┌─────┐
    │  4  │     │  5  │
    └─────┘     └─────┘
    해결 과제    조언
```

**위치 의미**:
| 위치 | 이름 | 의미 |
|------|------|------|
| 1 | 나의 마음 | 나의 감정과 태도 |
| 2 | 상대방의 마음 | 상대방의 감정과 태도 |
| 3 | 현재 관계 | 두 사람 사이의 현재 역학 |
| 4 | 해결해야 할 과제 | 관계 발전을 위해 극복할 것 |
| 5 | 조언 | 관계를 위한 가이드 |

---

### ⚖️ 선택 스프레드 (Choice Spread)

> 두 가지 선택지 사이에서 고민할 때 사용합니다.

**카드 수**: 5장  
**추천 상황**: 의사결정, 두 가지 옵션 비교, 진로 선택

```
         ┌─────┐
         │  1  │
         └─────┘
         현재 상황
         
    ┌─────┐     ┌─────┐
    │  2  │     │  4  │
    └─────┘     └─────┘
    선택 A      선택 B
    
    ┌─────┐     ┌─────┐
    │  3  │     │  5  │
    └─────┘     └─────┘
    A의 결과    B의 결과
```

**위치 의미**:
| 위치 | 이름 | 의미 |
|------|------|------|
| 1 | 현재 상황 | 선택을 해야 하는 배경 |
| 2 | 선택 A | 첫 번째 옵션의 에너지 |
| 3 | A의 결과 | 선택 A를 택했을 때 결과 |
| 4 | 선택 B | 두 번째 옵션의 에너지 |
| 5 | B의 결과 | 선택 B를 택했을 때 결과 |

---

### 🍀 켈틱 크로스 (Celtic Cross)

> 가장 전통적이고 상세한 분석을 제공하는 스프레드입니다.

**카드 수**: 10장  
**추천 상황**: 복잡한 상황, 깊은 분석, 인생의 중요한 결정

```
                        ┌─────┐
                        │ 10  │ 최종 결과
                        └─────┘
                        ┌─────┐
                        │  9  │ 희망과 두려움
                        └─────┘
    ┌─────┐             ┌─────┐
    │  4  │   ┌───┐     │  8  │ 주변 환경
    │과거 │ ──│1·2│──   └─────┘
    └─────┘   └───┘     ┌─────┐
              ┌─────┐   │  7  │ 자신
    ┌─────┐  │  6  │    └─────┘
    │  5  │  │미래 │
    │가능성│  └─────┘
    └─────┘
    ┌─────┐
    │  3  │
    │근본 │
    └─────┘
```

**위치 의미**:
| 위치 | 이름 | 의미 |
|------|------|------|
| 1 | 현재 상황 | 지금 처한 상황의 핵심 |
| 2 | 도전/장애물 | 현재 상황을 가로막는 것 |
| 3 | 근본 원인 | 상황의 뿌리, 무의식적 요인 |
| 4 | 최근 과거 | 바로 직전에 일어난 일 |
| 5 | 목표/가능성 | 의식적인 목표, 최선의 결과 |
| 6 | 가까운 미래 | 곧 다가올 상황 |
| 7 | 자신 | 상황에서의 나의 역할과 태도 |
| 8 | 주변 환경 | 외부 영향, 타인의 관점 |
| 9 | 희망과 두려움 | 내면의 기대와 걱정 |
| 10 | 최종 결과 | 상황의 결론 |

## 카테고리별 추천 스프레드

질문 카테고리에 따라 적합한 스프레드가 다릅니다:

| 카테고리 | 추천 스프레드 |
|----------|---------------|
| 💕 사랑 | 연애 스프레드, 쓰리카드, 켈틱 크로스 |
| 💼 일/커리어 | 쓰리카드, 선택 스프레드, 켈틱 크로스 |
| 🤝 인간관계 | 연애 스프레드, 쓰리카드, 선택 스프레드 |
| 💰 재정 | 쓰리카드, 선택 스프레드, 원카드 |
| 🌿 건강 | 원카드, 쓰리카드 |
| 🔮 일반 | 원카드, 쓰리카드, 켈틱 크로스 |

## 스프레드 데이터 구조

```typescript
interface SpreadPosition {
    id: number;
    name: string;
    meaning: string;
    x: number;  // 상대 위치 (0-100)
    y: number;
}

interface Spread {
    id: string;
    name: string;        // 영문명
    nameKr: string;      // 한글명
    description: string; // 설명
    cardCount: number;   // 카드 수
    positions: SpreadPosition[];
}
```

## 커스텀 스프레드 추가

`src/data/spreads.ts`에서 새로운 스프레드를 추가할 수 있습니다:

```typescript
export const myCustomSpread: Spread = {
    id: 'custom',
    name: 'Custom Spread',
    nameKr: '나만의 스프레드',
    description: '나만의 특별한 스프레드',
    cardCount: 4,
    positions: [
        { id: 1, name: 'position1', meaning: '첫 번째 위치', x: 25, y: 50 },
        { id: 2, name: 'position2', meaning: '두 번째 위치', x: 50, y: 25 },
        { id: 3, name: 'position3', meaning: '세 번째 위치', x: 50, y: 75 },
        { id: 4, name: 'position4', meaning: '네 번째 위치', x: 75, y: 50 },
    ]
};

// spreads 배열에 추가
export const spreads: Spread[] = [
    // ... 기존 스프레드
    myCustomSpread
];
```

## 스프레드 선택 화면
